    • sampgdk

    sampgdk allows you to write SA:MP game modes in C/C++

    Project maintained by Zeex Hosted on GitHub Pages — Theme by mattgraham

    Downloads

    You can download sampgdk development files and pre-compiled binaries for Linux and Windows here.

    Examples

    There are a few example plugins available in the source tree:

    • helloworld

      Hello World game mode that shows how to initialize sampgdk and add basic things like timers and commands.

    • lvdm

      A C++ port of the LVDM MoneyGrub game mode.

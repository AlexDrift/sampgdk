About

sampgdk is a library that allows one to write game modes for SA:MP in C++ or plain C rather than in Pawn. It is basically a layer on top of the official plugin SDK that provides an ability for plugins to call server's native functions and react on various events (callbacks) just like Pawn scripts do.

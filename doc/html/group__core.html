bool sampgdk_Load ( void **  ppData)

Initializes the library.

When called for the first time, sets up internal global state and registers the calling plugin for receiving events. All subsequent calls are ignored until sampgdk_Unload() is called.

This function is intended to be used in Load(). For example:

PLUGIN_EXPORT bool PLUGIN_CALL Load(void **ppData) {
...
return sampgdk_Load(ppData);
}
Parameters
ppDataA pointer to the SA-MP plugin data passed to Load().
Returns
Returns true on success and false otherwise.

sampgdk_Unload()

bool OnPublicCall ( AMX *  amx,
const char *  name,
cell *  params 
)

Called on every public function call.

This is the so called "public filter". It is called whenever the server calls to amx_Exec(). The return value controls whether the callback function corresponding to the public is executed in this plugin.

Parameters
amxThe AMX instance on which the public function is called.
nameThe name of the function.
paramsThe function's arguments as stored on the AMX stack, with the params[0] equal to the number of arguments multiplied by sizeof(cell).
Returns
If returns true the callback is executed, otherwise it is ignored.

int sampgdk_init ( void **  plugin_data)

Initializes the library and the calling plugin.

When called for the first time, this function sets up the internal global structures and registers the calling plugin for receiving certain events e.g. SA-MP callbacks. All suebsequent calls are ignored.

If for some reason this function fails it additionally may output an error message to the serrver log describing the error.

Parameters
plugin_dataA pointer to the SA-MP plugin data passed to Load().
Returns
A non-negative value on success or an error code on failure. The error code can be converted to a string using strerror().

cell sampgdk_invoke_native ( AMX_NATIVE  native,
const char *  format,
  ... 
)

Invokes a native function with the specified arguments.

The argument types are specified via format where each character, or specifier, corresponds to a single argument.

The following format specifiers are supported:

Specifier C/C++ type Description
i int an integer value
d int an integer value (same as 'i')
b bool a boolean value
f double a floating-point value
r const cell * a const reference
R cell * a non-const reference (output)
s const char * a string of character
S char * a string of character (output)
Note
For the 'S' specifier the argument next to it specifies the maximum length of the string. This convention is followed by all SA-MP functions.
Parameters
nativeA pointer to the native function.
formatA format string specifying the types of the arguments.
...The arguments themselves.
Returns
The value returned by the function.
